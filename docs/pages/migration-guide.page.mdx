Migration Guides for SemVer majors.

For SemVer minor changes, see: [CHANGELOG.md](https://github.com/brillout/vite-plugin-ssr/blob/master/CHANGELOG.md).

## 0.4

**Breaking**

 1. Update Vite to `>=2.9.6`. (Vite-plugin-ssr doesn't work with older versions.)
 1. Both `useClientRouter()` (Client Routing) and `getPage()` (Server Routing) are deprecated. Use `export { render }` in `.page.client.js` instead.
 2. If you use pre-render options or base URL options: they are now defined in `vite.config.js`.
 3. If you do `export default Page`: do `export { Page }` instead.

Most apps will only require 1.

**Non-breaking**

There are other changes but these are non-breaking: follow the warnings to (progressively) update to the new 0.4 interface.
