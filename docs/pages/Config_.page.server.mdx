```ts
// +config.ts

export default {
  // ...
} satisfies Config
```

## Configs

- **`Page`** (`unknown`): the root UI component of the page.
- **`route`** (`string | Function`): the page's route. See <Link href="/routing" />.
- **`passToClient`** (`string[]`): determine what `pageContext` properties are sent to the client-side. See <Link href="/passToClient" />.
- **`clientRouting`** (`boolean`): Enable Client Routing. See <Link href="/clientRouting" />.
- **`meta`** (`Meta`): Create new or modify existing configurations. See <Link href="/meta" />.
- **`prefetchStaticAssets`** (`PrefetchStaticAssets`): Prefetch links. See <Link href="/clientRouting#link-prefetching" />.
- **`client`** (`string`): Additional client entry.
- **`hydrationCanBeAborted`** (`boolean`):  Whether your UI framework (React/Vue/Solid/...) allows the page's hydration to be aborted. See <Link href="/clientRouting" />.
- **`iKnowThePerformanceRisksOfAsyncRouteFunctions`** (`boolean`): Enable async Route Functions. See <Link href="/route-function#async" />.

## Hooks

- **`onRenderHtml`** [server-side]: Hook called when page is rendered to HTML on the server-side. See <Link href="/onRenderHtml" />.
- **`onRenderClient`** [client-side]: Hook called when page is rendered on the client-side. See <Link href="/onRenderClient" />.
- **`onBeforeRender`** [server-side]: Hook called before the page is rendered, usually for fetching data. See <Link href="/onBeforeRender" />.
- **`onHydrationEnd`** [client-side]: Hook called after the page is hydrated. See <Link href="/clientRouting" />.
- **`onPageTransitionStart`** [client-side]: Hook called before the user navigates to a new page. See <Link href="/clientRouting" />.
- **`onPageTransitionEnd`** [client-side]: Hook called after the user navigates to a new page. See <Link href="/clientRouting" />.
- **`onPrerenderStart`** [server-side]: Page Hook called when pre-rendering starts. See <Link href="/onPrerenderStart" />.
- **`onBeforePrerenderStart`** [server-side]: Global Hook called before the whole pre-rendering process starts. See <Link href="/onBeforePrerenderStart" />.
- **`onBeforeRoute`** [server (and client-side)]: Hook called before the URL is route to a page. See <Link href="/onBeforeRoute" />.

## Custom

New configurations can be created by using <Link text="meta" href="/meta" />.
