```ts
// +config.ts

// The list is also available over TypeScript
import type { Config } from 'vite-plugin-ssr/types'

export default {
  // ...
} satisfies Config
```

## Configs

- **`Page`** (`unknown`): the root UI component of the page.
- **`route`** (`string | Function`): the page's route. See <Link href="/routing" />.
- **`passToClient`** (`string[]`): determine what `pageContext` properties are sent to the client-side. See <Link href="/passToClient" />.
- **`clientRouting`** (`boolean`): Enable Client Routing. See <Link href="/clientRouting" />.
- **`meta`** (`Meta`): Create new or modify existing configurations. See <Link href="/meta" />.
- **`prefetchStaticAssets`** (`PrefetchStaticAssets`): Prefetch links. See <Link href="/clientRouting#link-prefetching" />.
- **`client`** (`string`): Additional client entry.
- **`hydrationCanBeAborted`** (`boolean`):  Whether your UI framework (React/Vue/Solid/...) allows the page's hydration to be aborted. See <Link href="/clientRouting" />.
- **`iKnowThePerformanceRisksOfAsyncRouteFunctions`** (`boolean`): Enable async Route Functions. See <Link href="/route-function#async" />.

## Hooks

- [**`onRenderHtml`**](https://vite-plugin-ssr.com/onRenderHtml): <TextGray>[server-side]</TextGray> Hook called when page is rendered to HTML on the server-side.
- [**`onRenderClient`**](https://vite-plugin-ssr.com/onRenderClient): <TextGray>[client-side]</TextGray> Hook called when page is rendered on the client-side.
- [**`onBeforeRender`**](https://vite-plugin-ssr.com/onBeforeRender): <TextGray>[server-side]</TextGray> Hook called before the page is rendered, usually for fetching data.
- [**`onHydrationEnd`**](https://vite-plugin-ssr.com/clientRouting): <TextGray>[client-side]</TextGray> Hook called after the page is hydrated.
- [**`onPageTransitionStart`**](https://vite-plugin-ssr.com/clientRouting): <TextGray>[client-side]</TextGray> Hook called before the user navigates to a new page.
- [**`onPageTransitionEnd`**](https://vite-plugin-ssr.com/clientRouting): <TextGray>[client-side]</TextGray> Hook called after the user navigates to a new page.
- [**`onPrerenderStart`**](https://vite-plugin-ssr.com/onPrerenderStart): <TextGray>[server-side]</TextGray> Page Hook called when pre-rendering starts.
- [**`onBeforePrerenderStart`**](https://vite-plugin-ssr.com/onBeforePrerenderStart): <TextGray>[server-side]</TextGray> Global Hook called before the whole pre-rendering process starts.
- [**`onBeforeRoute`**](https://vite-plugin-ssr.com/onBeforeRoute): <TextGray>[server (and client-side)]</TextGray> Hook called before the URL is route to a page.
