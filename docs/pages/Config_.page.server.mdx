```ts
// +config.ts

export default {
  // ...
} satisfies Config
```

## Configs

- **`Page`** (`unknown`): the root UI component of the page.
- **`route`** (`string | Function`): the page's route. See <Link href="/routing" />.
- **`passToClient`** (`string[]`): determine what `pageContext` properties are sent to the client-side. See <Link href="/passToClient" />.
- **`clientRouting`** (`boolean`): Enable Client Routing. See <Link href="/clientRouting" />.
- **`meta`** (`Meta`): Create new or modify existing configurations. See <Link href="/meta" />.
- **`prefetchStaticAssets`** (`PrefetchStaticAssets`): Prefetch links. See <Link href="/clientRouting#link-prefetching" />.
- **`client`** (`string`): Additional client entry.
- **`hydrationCanBeAborted`** (`boolean`):  Whether your UI framework (React/Vue/Solid/...) allows the page's hydration to be aborted. See <Link href="/clientRouting" />.
- **`iKnowThePerformanceRisksOfAsyncRouteFunctions`** (`boolean`): Enable async Route Functions. See <Link href="/route-function#async" />.

## Hooks

- [**`onRenderHtml`** [server-side]](https://vite-plugin-ssr.com/onRenderHtml): Hook called when page is rendered to HTML on the server-side.
- [**`onRenderClient`** [client-side]](https://vite-plugin-ssr.com/onRenderClient): Hook called when page is rendered on the client-side.
- [**`onBeforeRender`** [server-side]](https://vite-plugin-ssr.com/onBeforeRender): Hook called before the page is rendered, usually for fetching data.
- [**`onHydrationEnd`** [client-side]](https://vite-plugin-ssr.com/clientRouting): Hook called after the page is hydrated.
- [**`onPageTransitionStart`** [client-side]](https://vite-plugin-ssr.com/clientRouting): Hook called before the user navigates to a new page.
- [**`onPageTransitionEnd`** [client-side]](https://vite-plugin-ssr.com/clientRouting): Hook called after the user navigates to a new page.
- [**`onPrerenderStart`** [server-side]](https://vite-plugin-ssr.com/onPrerenderStart): Page Hook called when pre-rendering starts.
- [**`onBeforePrerenderStart`** [server-side]](https://vite-plugin-ssr.com/onBeforePrerenderStart): Global Hook called before the whole pre-rendering process starts.
- [**`onBeforeRoute`** [server (and client-side)]](https://vite-plugin-ssr.com/onBeforeRoute): Hook called before the URL is route to a page.

## Custom
