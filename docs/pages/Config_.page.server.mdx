Configurations defined by <Link text={<code>+config.js</code>} href="/config" />.

```ts
// +config.ts

// The list is also available over TypeScript
import type { Config } from 'vite-plugin-ssr/types'

export default {
  // ...
} satisfies Config
```

## Configs

**Basics**

- [**`Page`**](/Page) (`unknown`): The root UI component of the page.
- [**`route`**](/routing) (`string | Function`): The page's route.
- [**`prerender`**](/prerender): Whether to <Link text="pre-render" href="/preredering" /> the page.
- [**`client`**](/client) (`string`): Additional client entry.
- [**`prefetchStaticAssets`**](/clientRouting#link-prefetching) (`PrefetchStaticAssets`): Prefetch links.
- [**`iKnowThePerformanceRisksOfAsyncRouteFunctions`**](/route-function#async) (`boolean`): Enable async Route Functions.

**Advanced**

> The following configurations are usually only used by <Link href="/vike-packages" noBreadcrumb />. The vast majority of users don't need to know about these.

- [**`meta`**](/meta) (`Meta`): Create new or modify existing configurations.
- [**`passToClient`**](/passToClient) (`string[]`): Determines what `pageContext` properties are sent to the client-side.
- [**`clientRouting`**](/clientRouting) (`boolean`): Enable Client Routing.
- [**`hydrationCanBeAborted`**](/clientRouting#usage-options) (`boolean`):  Whether the <Link text="UI framework" href="/ui-framework" /> allows hydrations to be aborted.

## Hooks

- [**`onRenderHtml`**](/onRenderHtml): <TextGray>[server-side]</TextGray> Called when page is rendered to HTML on the server-side.
- [**`onRenderClient`**](/onRenderClient): <TextGray>[client-side]</TextGray> Called when page is rendered on the client-side.
- [**`onBeforeRender`**](/onBeforeRender): <TextGray>[server-side]</TextGray> Called before the page is rendered, usually for fetching data.
- [**`onHydrationEnd`**](/clientRouting#usage-options): <TextGray>[client-side]</TextGray> Called after the page is hydrated.
- [**`onPageTransitionStart`**](/clientRouting#usage-options): <TextGray>[client-side]</TextGray> Called before the user navigates to a new page.
- [**`onPageTransitionEnd`**](/clientRouting#usage-options): <TextGray>[client-side]</TextGray> Called after the user navigates to a new page.
- [**`onPrerenderStart`**](/onPrerenderStart): <TextGray>[server-side]</TextGray> Page-specific hook called when pre-rendering starts.
- [**`onBeforePrerenderStart`**](/onBeforePrerenderStart): <TextGray>[server-side]</TextGray> Global hook called before the whole pre-rendering process starts.
- [**`onBeforeRoute`**](/onBeforeRoute): <TextGray>[server (and client-side)]</TextGray> Called before the URL is routed to a page.

## Custom

New configurations can be created by using <Link text="meta" href="/meta" />.
