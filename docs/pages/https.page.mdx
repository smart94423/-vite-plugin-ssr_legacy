From a production server perspective, `vite-plugin-ssr` is only a server middleware and there is nothing to take into consideration.

To use HTTPS also for dev, we need to pass the HTTPS certificates to Vite's dev server. For example:
 - [github.com/aral/vite-plugin-ssr-with-tls](https://github.com/aral/vite-plugin-ssr-with-tls)
