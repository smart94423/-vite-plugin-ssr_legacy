From a perspective of Docker, a `vite-plugin-ssr` app is simply a Node.js server, so we can use any Node.js server Docker configuration.

If we want to use Docker for dev, we need to:
 - Properly configure the HMR port
 - We may need to use polling: [vite.config.js#server.watch.usePolling](https://vitejs.dev/config/#server-watch) or [StackOverflow - Chokidar isn't picking up file changes inside docker container](https://stackoverflow.com/questions/38318188/chokidar-isnt-picking-up-file-changes-inside-docker-container/69166925#69166925).
