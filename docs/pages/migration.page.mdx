## 0.4

**Non-breaking**

Most changes are non-breaking: follow the warnings to (progressively, at your own pace) update to the new 0.4 interface.

**Breaking**

 1. Use Vite `>=2.9.6`.
 1. Deprecated: `useClientRouter()` (Client Routing) and `getPage()` (Server Routing).
 1. Base URL option defined only in `vite.config.js`.
 1. Pre-render options defined in `vite.config.js`.
 1. Deprecated: `export default Page`. (Use `export { Page }` instead.)
