> **What are active links?** It's the practice of visually highlighting the current page in the navigation.
> For example, this page is called "Active Links" and is highlighted with a gray background in the navigation on the left of this website.
> The link is said to be "active".

To implement active links:
 - We create a new component `<Link>`.
 - We make [`pageContext.urlPathname`](/pageContext) available to our new component `<Link>`. See <Link href="/pageContext-anywhere"/>.

Our `<Link>` component checks whether `const isActive = href === pageContext.urlPathname` and sets a CSS class accordingly `<a class="is-active">`.

> If we <Link text="render our page as SPA" href="/render-modes" />, then we can use `window.location.pathname` instead of `pageContext.urlPathname`.
