Environment: `Node.js` (and `Browser` if using Client-Side Routing)<br/>
[Ext Glob](https://github.com/micromatch/micromatch#extglobs): `/**/_default.page.route.*([a-zA-Z0-9])`

The `_default.page.route.js` file has two exports that allows us to control the routing process:
 - [`export { filesystemRoutingRoot }`](filesystemRoutingRoot)
 - [`export { onBeforeRoute }`](onBeforeRoute)
