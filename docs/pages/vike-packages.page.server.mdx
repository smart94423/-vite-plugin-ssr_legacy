> Vike is the upcoming new name for vite-plugin-ssr, see [#736](https://github.com/brillout/vite-plugin-ssr/issues/736).

The `vike-*` packages automatically integrate <Link text="UI frameworks" href="/ui-framework" /> with Vike (using the <Link text="V1 design" href="/migration/v1-design" />).

 - [`vike-react`](https://github.com/vikejs/vike-react)
 - [`vike-vue`](https://github.com/vikejs/vike-vue)
 - [`vike-solid`](https://github.com/magne4000/vike-solid)

> These packages are going to get out of beta soon.

> You can use [Bati](https://batijs.github.io/) to easily scaffold a Vike app using one of the
> `vike-*` packages.
