> Vike is the upcoming new name for vite-plugin-ssr, see [#736](https://github.com/brillout/vite-plugin-ssr/issues/736).

The `vike-*` packages automatically integrate <Link text="UI frameworks" href="/ui-framework" /> with Vike.

 - [`vike-react`](https://github.com/vikejs/vike-react)
 - [`vike-vue`](https://github.com/vikejs/vike-vue)
 - [`vike-solid`](https://github.com/magne4000/vike-solid)

<Construction>These packages are work-in-progress.</Construction>

> You can use [Bati](https://batijs.github.io/) to easily scaffold a project using `vite-plugin-ssr` and one of the
> `vike-*` packages. This uses the [V1 design](/migration/v1-design).
