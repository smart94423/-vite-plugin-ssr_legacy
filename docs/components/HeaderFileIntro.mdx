Vite-plugin-ssr transforms `.h.js` files such as this:

```js
// /pages/some-page/+config.h.js

import Page from './some-file'
import onBeforeRender from './some-other-file'

export default {
  Page,
  onBeforeRender
}
```

Into:

```js
// /pages/some-page/+config.h.js

const Page = 'import:./some-file';
const onBeforeRender = 'import:./some-other-file';

export default {
  Page,
  onBeforeRender
}
```

> `.h.js` is optional syntatic sugar for improved IntelliSense (go-to-definition, auto-import, type-safety, refactoring, ...). It's only meant for config files (any `.h.js` file should be a `+config.h.js` file).
